# Burning
버닝은 후잉 연동 투자 자산 관리 시스템입니다.
후잉 가계부 데이터를 활용하여 투자 자산을 더 잘 관리하게 도와주는 웹 서비스입니다.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
